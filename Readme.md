# ProDER: Prototype-Guided Dynamic Enhancement Representation for PNEN Grading

## Overview

This repository contains the implementation of **ProDER**, an interpretable deep learning framework for preoperative grading of Pancreatic Neuroendocrine Neoplasms (PNENs) using dual-phase contrast-enhanced CT imaging.

### Key Features

- **Soft-Attention Mechanism**: Guides the model toward tumor-salient regions using a frozen segmentation prior
- **Explicit Dynamic Modeling**: Captures arterial-to-venous enhancement kinetics through Siamese encoding and feature subtraction
- **Prototype-Based Classification**: Provides interpretable predictions through learnable grade-specific prototype directions
- **Clinical Alignment**: Mimics pathological diagnostic logic for Ki-67 hotspot assessment
